QUANTUM ! CRYPTOSYSTEM

Secure File Storage Using Hybr!d Cryptography

Objective: To achieve a platform users can believe in for storing files on the cloud using hybrid cryptography.

Methodology:
  1. Load the file on the server. It can be any type of file.
  2. Dividing the uploaded file into N parts depending on the fixed block size.
  3. Encrypting all the sub-files using one of the selected algorithms (the algorithm changes with every part in a round-robin fashion).
  4. The keys for cryptographic algorithms used are stored in a file, secured using a different cryptographic algorithm. The key for this algorithm is provided to the user as a public key, which needs to be transferred safely to the receiver.

To restore:

  1. Load the key (.pem file) on the server. Ask it from your sender.
  2. Decrypt the keys of the algorithms.
  3. Decrypt all the N subfiles using the same algorithms used to encrypt them.
  4. Combine all the N subfiles to form the original file and provide it to the user for downloading.
